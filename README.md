# northwind_task

# Task: Write a SQL query that returns the following information for each category:
•	CategoryName/
•	Total Sales
•	AverageOrderValue
•	AverageTimeToShip (in days)
•	Top5Ratio (total sales of the top 5 products to the total sales of all products in the category)
•	AverageOrderFrequency (number of orders per customer)
•	The result should be sorted by Total Sales in descending order

# Database
The project uses the Northwind database adapted for PostgreSQL: https://github.com/pthom/northwind_psql
The schema and sample data were restored and queried using PostgreSQL (via pgAdmin 4)
